# **Индивидуальная часть проекта. Галкина Мария, группа №4**

## Введение:
> Целью работы над проектом является поиск и изучение консервативных участков генома Z-ДНК в геномах прокариот и эукариот. В процессе выполнения проекта будут получены важные практические навыки биоинформатики - работа в командной строке с утилитой bedtools, визуализация данных, конвертация координат участков между разными версиями генома (разными организмами) и т.п.
